# Wet-lab rationale (review-gated, non-ordering)

This directory holds **review-gated, non-ordering** wet-lab rationale drafts:
the scientific reasoning behind why a candidate or family might warrant an assay,
written for human review.

It is currently **empty by design**. Protean's wet-lab loop is review-gated and
dormant pending trusted, human-reviewed assay labels. No content in this archive
places, accepts, or authorizes any provider order. When review-ready rationale is
published, drafts land under `review-ready/` and superseded drafts move to
`archived/`.

Nothing here is a wet-lab order, a quote acceptance, or a provider submission.
See [`../BOUNDARY.md`](../BOUNDARY.md).
